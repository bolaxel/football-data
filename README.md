# Real Data for Football Analysis

Only CSV files scraped from different sites.

If you want to get data from **FBREF**, **FOTMOB**, **SOFASCORE**, **UNDERSTAT** check this repository [scraping football websites](https://github.com/axelbol/scraping-football-sites).

## Repository details

__What can you get from wcup26/fotmob_match_stats.csv?__

This CSV file contains detailed data from all matches in the FIFA 2026 World Cup.

- **Big chances**
- **Big chances missed**
- **Fouls**
- **Total shots**
- **xG**
- **xG open play**
- **xG set play**
- **xG non penalty**
- **xGOT home**
- More ...
